# RakkaRecorder

RakkaRecorder - Free PerfRecorder

这是一个免费的 Android 应用，可以测量并记录手机游戏和应用的性能。
可以同时查看 FPS、CPU/GPU 使用率、温度、电池功耗等信息。

支持 **Root / Shizuku** 运行。

## 应用简介

RakkaRecorder 可以在玩游戏时实时显示 FPS、温度等信息，
并且可以把所有性能数据完整记录下来。

主要功能：

-   游戏过程中悬浮窗显示 FPS
-   实时监测 CPU / GPU / RAM / 电池状态
-   按会话记录所有测量数据
-   通过图表查看历史数据
-   导出 CSV 进行外部分析

------------------------------------------------------------------------

## 系统要求

  项目   内容
  ------ ------------------------------
  系统   Android 14+ (API 34+)
  权限   Root 或 Shizuku
  其他   悬浮窗权限、使用情况访问权限

必须拥有 **Root 或 Shizuku** 其中之一，否则部分数据无法获取。

------------------------------------------------------------------------

## 使用方法

1.  启动应用
2.  检查权限是否全部授予
3.  选择需要监测的应用（可选）
4.  点击 **开始监测**
5.  打开游戏或应用
6.  完成后返回 RakkaRecorder 点击 **停止并保存**
7.  在记录页面查看结果

## 可监控项目

### FPS

-   FPS
-   总帧数

### CPU

-   总 CPU 使用率
-   每核心使用率
-   集群频率（Little / Big / Prime）
-   CPU 温度

### GPU

-   GPU 使用率
-   GPU 频率
-   GPU 温度

### 内存

-   RAM 使用情况

### 电池

-   电池电流 (mA)
-   电池电压 (mV)
-   电量 (%)
-   电池温度
-   功耗 (mW)
-   每帧功耗 (mW/fps)

### 其他

-   机身表面温度

------------------------------------------------------------------------

## FPS 监测机制

FPS 使用四级回退机制：

1.  SurfaceFlinger --timestats
2.  dumpsys gfxinfo PROFILEDATA
3.  SurfaceFlinger --latency
4.  gfxinfo Total frames

------------------------------------------------------------------------

## 悬浮窗显示

可自定义：

-   显示位置
-   字体大小
-   背景透明度

支持显示：

-   FPS
-   CPU / GPU 信息
-   RAM 使用情况
-   电池信息
-   功耗
-   温度

------------------------------------------------------------------------

## 记录与图表

记录会话包括：

-   应用名称
-   时间
-   持续时长
-   样本数量

图表显示 **min / avg / max**。

支持 CSV 导出（1秒=1条记录）。

**Q: 没有 Root 可以使用吗？**\

**Q: FPS 显示为 0**\
A: 请确认 Root / Shizuku 权限正确。
